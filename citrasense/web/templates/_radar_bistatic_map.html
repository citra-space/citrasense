{# Top-down plan-view plot for the bistatic geometry: - Receiver at the centre (gold dot + "RX" label). - FM transmitter stations at their real lat/lon offsets (cyan dots + callsign labels). - All currently-tracked satellites as ground-projected sub-points, with a short trail per ``sat_uuid`` (oldest faded). Data sources (from the Alpine store's enriched ``sensor``): - ``sensor.radar.receiver`` -- ``{lat_deg, lon_deg, alt_m}``, cached from the most recent observation's ``geometry.receiver``. - ``sensor.radar.stations.stations`` -- the list of :class:`StationInfo` dicts pr_sensor publishes on ``radar.sensor.{id}.stations``. - ``$store.citrasense.radarDetections[sensor.id]`` -- the same ring-buffered slim dicts the Range-Doppler plot consumes. Each now also carries ``az_deg`` / ``el_deg`` / ``alt_km`` / ``station`` from the sensor's slim projection. Projection (flat-earth; accurate to <<1% for the LEO ranges we care about): ground_range_km = range_km * cos(el_rad) dE = ground_range_km * sin(az_rad) (east, +x) dN = ground_range_km * cos(az_rad) (north, -y in SVG) dN_station = (sta.lat - rx.lat) * 111.32 dE_station = (sta.lon - rx.lon) * 111.32 * cos(rx.lat) Bounds auto-fit the max |dE|, |dN| across the receiver, all stations, and all tracked sat points; clamped to a 300 km floor so a single nearby pass doesn't over-zoom the view.
